Generated Reports

This document provides an overview of the various reports that are automatically generated by Maven . Each report is briefly described below.

Overview

Document Description
JavaDoc HTML HTML documents of JavaDoc.
JavaDoc PDF PDF document of JavaDoc (com.tarsec.javadoc.pdfdoclet).
JavaDoc UML (yUML) UML diagrams in HTML JavaDoc (ydoc.doclets.YStandard).
JavaDoc UML (plantUML) UML diagrams in HTML JavaDoc (org.umlgraph.doclet).
JavaDoc UML (umlgraph) UML diagrams in HTML JavaDoc (org.umlgraph.doclet).
JavaDoc DocBook JavaDoc DocBook
JavaDoc ApiGuide JavaDoc ApiGuide.
JavaDoc XHTML JavaDoc XHTML.
JavaDoc LaTeX JavaDoc LaTeX.
Source Xref HTML based, cross-reference version of Java source code.
Changes Changes report on releases of this project.
