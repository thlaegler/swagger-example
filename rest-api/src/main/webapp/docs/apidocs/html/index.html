post /somethings
Add a new Something (add)
Add a new Something.

Consumes

This API call consumes the following media types via the Content-Type request header:
  • application/json
  • application/vnd.de.tk.tkapp.v1+json
  • application/xml

Request body

body (required)
Body Parameter — Something object that needs to be added to database.

Return type

Something

Example data

Content-Type: application/json
{
  "name" : "aSomething",
  "id" : 1,
  "otherObject" : {
    "name" : "aOtherObject",
    "id" : 1
  }
}

Example data

Content-Type: application/xml

  1
  aSomething

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json
  • application/vnd.de.tk.tkapp.v1+json
  • application/xml

Responses

200

New Something successfully added. Something

201

New Something successfully added.

400

Invalid Something supplied.

405

Validation exception.

0

An unexpected error occured. Error
