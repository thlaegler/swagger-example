This is the API

add

Add a new Something.

URL

http://localhost:8080/tkapp/somethings

HTTP Method

POST

Response Type

Something

Parameters

  • Body: body Something(Something)

    Something object that needs to be added to database.

findAll

Get all existing Somethings.

URL

http://localhost:8080/tkapp/somethings

HTTP Method

GET

Response Type

List

Parameters

findById

Find an existing Something from database by ID.

URL

http://localhost:8080/tkapp/somethings/{id}

HTTP Method

GET

Response Type

Something

Parameters

  • Path: id Long

    ID of Something that needs to be fetched.

remove

Remove an existing Something from database.

URL

http://localhost:8080/tkapp/somethings

HTTP Method

DELETE

Response Type

Parameters

  • Body: body Something(Something)

    Something object that needs to be removed from database.

update

Update an existing Something.

URL

http://localhost:8080/tkapp/somethings

HTTP Method

PUT

Response Type

Something

Parameters

  • Body: body Something(Something)

    Something object that needs to be added to database.
